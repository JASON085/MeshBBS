# 維護紀錄 b0603i

日期：2026-05-20

## 修改內容

- Android Server 行為再往 PC Server 對齊，LIST 回傳不再額外帶線上使用者清單，減少封包大小。
- Server 請求處理改成單一背景佇列，避免阻塞 Meshtastic 封包接收。
- Python Server module 改為快取，減少每次 BBS 指令的固定呼叫開銷。
- 版本號更新為 `b0603i`。

- `python -m py_compile android/MeshtasticBBS/app/src/main/python/meshbbs_android_server.py` 通過。
- `build_apk.ps1 -Variant server -BuildType release` 成功。
- 輸出 APK：
  - `android/MeshtasticBBS/app/build/outputs/apk/server/release/MeshBBS-server-b0603i.apk`
