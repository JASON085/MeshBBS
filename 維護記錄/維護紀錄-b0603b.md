# 維護紀錄 b0603b

日期：2026-05-20

## 本次修改

- 製作新版 Android Server APK：`MeshBBS-server-b0603b.apk`
- Android Server Python 核心改為明確使用 app `filesDir` 內的 `meshbbs_android_server.db`
- 新增 Android Server 資料庫備份功能
  - 於 Server 主機頁加入 `備份 DB` 按鈕
  - 備份檔輸出到 app 外部文件資料夾下的 `meshbbs_backups`
  - 主機頁可顯示備份資料夾、最新備份、最近備份清單
- Android Server dashboard 補強
  - 管理員清單
  - 近期文章清單
  - 備份資訊
- 啟動 bridge 時會先同步 storage 路徑，避免資料庫位置漂移
- 若主資料庫不存在，會嘗試從最近一次備份還原

## 與 PC Server 對齊狀況

- 先把 Android Server 的資料庫管理、管理員摘要、近期文章摘要補上
- 目前已更接近 PC Server 的主機管理資訊，但完整的管理員 CRUD、看板 CRUD、使用者停權/改密碼等互動式後台 UI 尚未全部搬到 Android 端

## 驗證

- `python -m py_compile android/MeshtasticBBS/app/src/main/python/meshbbs_android_server.py`
- `build_apk.ps1 -Variant server -BuildType release` 成功
- 產出：
  - `android/MeshtasticBBS/app/build/outputs/apk/server/release/MeshBBS-server-b0603b.apk`
