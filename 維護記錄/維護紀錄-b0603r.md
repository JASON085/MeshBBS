# 維護紀錄 b0603r

日期：2026-05-21

## 修改內容

- 修正 `MeshBBS` 站內通訊頁可能因訊息列表 key 重複而報錯退出的問題。
- 移除 `MeshChatScreen.kt` 對訊息內容拼接的自訂 key，避免相同發送者、時間、內容的訊息造成 Compose 清單崩潰。
- 版本號更新為 `b0603r`，避免與既有 APK 重複。

## 驗證

- Client APK 建置成功：
  - `android/MeshtasticBBS/app/build/outputs/apk/client/release/MeshBBS-b0603r.apk`
  - `android/MeshtasticBBS/apk_archive/MeshBBS-b0603r.apk`

## 修正時發現的問題

- 站內通訊清單先前用 `from + time + text` 當 key，當本機先插入訊息、之後又收到同內容回傳時，容易產生重複 key。
