# 維護紀錄 b0604a

日期：2026-05-21

## 修改摘要

- 修正 Android Server 看板管理的「編輯看板」版主名稱欄位，改為可正常展開下拉選單，能直接選擇已註冊使用者為版主。
- Client 發表文章與回覆文章送出後，立即鎖定標題與內容輸入區，避免使用者在傳送途中再次修改內容。
- Client 發文頁新增送出進度顯示，包含壓縮階段、分段發送階段與等待伺服器回應階段。
- 發文與回覆送出期間保持畫面常亮，避免 Meshtastic 傳輸較慢時螢幕熄滅中斷操作感受。
- Client 端文章與回覆改為先壓縮再分段傳送；Android Server 收到後會重組、解壓縮，再交由伺服器核心處理。
- 版本號更新為 `b0604a`。
