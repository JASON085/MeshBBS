# MeshBBS b0602r 維護紀錄

- 依據 b0602q 診斷結果，確認 `getNodes()` 會因 Meshtastic 新版 Parcelable 格式差異失敗，無法再依賴此路徑建立節點清單。
- 改為直接從 `NODE_CHANGE` 廣播與收到的 Meshtastic 封包來源節點補建節點清單，避免節點頁面維持空白。
- `NODE_CHANGE` 新增 Parcelable 與 bundle 掃描雙重解析，盡量抓出節點 ID 與名稱。
- Client 版本號更新為 `b0602r`，避免與既有 APK 重複。
- 已產出新版 Client APK：`android/MeshtasticBBS/apk_archive/MeshBBS-b0602r.apk`
