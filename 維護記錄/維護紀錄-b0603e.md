# 維護紀錄 b0603e

日期：2026-05-20

## 本次修改

- 製作新版 Android Server APK：`MeshBBS-server-b0603e.apk`
- 修正 Android Server 接收請求時的來源節點 ID 擷取
  - 優先取 `DataPacket.from`
  - 若取不到，再從 bundle/parcel 字串掃描 node id
- 修正 Android Server 私訊 `?` 無回應問題
  - 現在可直接在 Meshtastic APP 私訊 `?`
  - Server 會以一般文字訊息回傳簡要說明
- Android Server 現在不只接受 `BBS:REQ:` 封包，也會接住可解析的 plain text 請求

## 目的

- 讓 MeshBBS Client 登入時，Server 回應能正確送回原本的節點
- 讓使用者能先用 Meshtastic APP 私訊 `?` 快速驗證 Android Server 是否真的有收送正常

## 驗證

- `python -m py_compile android/MeshtasticBBS/app/src/main/python/meshbbs_android_server.py`
- `build_apk.ps1 -Variant server -BuildType release` 成功
- 產出：
  - `android/MeshtasticBBS/app/build/outputs/apk/server/release/MeshBBS-server-b0603e.apk`
  - `android/MeshtasticBBS/apk_archive/MeshBBS-server-b0603e.apk`
