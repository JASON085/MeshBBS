## 修改內容

- `MeshBBS` 站內通訊改為由使用者直接送出文字訊息，不再透過 Android Server 的 `CHAT` 協定回應，避免 server 在頻道內發出亂碼內容。

- 版本號更新為 `b0603u`，避免與既有 APK 重複。
