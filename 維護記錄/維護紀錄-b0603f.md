# 維護紀錄 b0603f

日期：2026-05-20

## 本次修改

- 製作新版 Android Server APK：`MeshBBS-server-b0603f.apk`
- Android Server 改為在前景服務啟動時取得 `PARTIAL_WAKE_LOCK`
- 停止 Server 或服務銷毀時會釋放喚醒鎖
- 新增 `省電白名單` 按鈕
  - 會嘗試開啟 Android 的忽略電池最佳化請求畫面
  - 讓使用者可把 Android Server 加入白名單
- Manifest 新增：
  - `android.permission.WAKE_LOCK`
  - `android.permission.REQUEST_IGNORE_BATTERY_OPTIMIZATIONS`
  - `foregroundServiceType="dataSync"`

## 目的

- 讓 Android Server 在手機螢幕關閉後仍能持續運作
- 降低 CPU 休眠或系統省電策略導致 Server 停止收送的機率

## 驗證

- `python -m py_compile android/MeshtasticBBS/app/src/main/python/meshbbs_android_server.py`
- `build_apk.ps1 -Variant server -BuildType release` 成功
- 產出：
  - `android/MeshtasticBBS/app/build/outputs/apk/server/release/MeshBBS-server-b0603f.apk`
  - `android/MeshtasticBBS/apk_archive/MeshBBS-server-b0603f.apk`

## 備註

- 即使已加喚醒鎖，不同品牌手機仍可能有額外背景限制
- 建議實機上再手動確認已加入省電白名單
