# 維護紀錄 b0603d

日期：2026-05-20

## 本次修改

- 製作新版 Android Server APK：`MeshBBS-server-b0603d.apk`
- Android Server 主機頁新增 `匯入 DB` 按鈕
- 可從 Android 裝置儲存空間選擇外部 `.db` 檔匯入
- 匯入前會先自動備份目前 Android Server 的資料庫
- 匯入後會自動對資料表做相容性補齊，方便接收 PC Server 的 DB
  - `users.banned`
  - `users.password_hash`
  - `users.is_web`
  - `boards.moderator_id`
  - `posts.push_count`
  - `post_pushes` table

## 匯入流程

- 在 Android Server 主機頁點 `匯入 DB`
- 選擇從 PC Server 複製過來的資料庫檔
- 匯入成功後會直接覆蓋目前 Android Server 使用中的主 DB
- 舊 DB 會先備份到 `meshbbs_backups`

## 驗證

- `python -m py_compile android/MeshtasticBBS/app/src/main/python/meshbbs_android_server.py`
- `build_apk.ps1 -Variant server -BuildType release` 成功
- 產出：
  - `android/MeshtasticBBS/app/build/outputs/apk/server/release/MeshBBS-server-b0603d.apk`
  - `android/MeshtasticBBS/apk_archive/MeshBBS-server-b0603d.apk`
