# MeshBBS b0602i 維護紀錄
- 新增 Android `Home` 入口，可在 Client 模式與 Android Server Host 模式之間切換，保留原本 PC Server 與 Android Client 路線。
- 新增 Android Server Host 第一版骨架：前台服務、Meshtastic App 綁定、Server 狀態畫面、Python `meshbbs_android_server.py` 核心與 SQLite 本機資料庫。
- Android Server Host 目前已可處理 `LOGIN`、`LOGOUT`、`LIST2`、`POSTS2`、`READ2`、`POST`、`REPLY`、`EDIT`、`DEL`、`DELREP`、`EDITREP`、`PUSH`、`SEARCH`、`CHAT` 等核心 BBS 指令。
- 建置時發現上一版號 `b0602h` 已存在於 `apk_archive/`，因此本次版本號改為 `b0602i` 後重新建置，避免 APK 版本撞號。
- 已用 `android/MeshtasticBBS/build_apk.ps1` 驗證 Android 專案可成功編譯；Chaquopy 已實際打包進 APK。
