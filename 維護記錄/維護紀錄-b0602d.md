# MeshBBS b0602d 維護紀錄

- 目標：對 MeshBBS Android App 做完整檢查並改善執行效率。
- 修正：
  - 文章列表載入加入同頁請求節流，避免滑動或重組時重複送出同一頁 `POSTS2`。
  - 下一頁載入只在非載入中觸發，降低重複分頁請求。
  - 文章開啟加入短時間重複點擊保護，避免同一篇文章連續送多次 `READ2`。
  - BBS 診斷紀錄限制最大長度，避免登入頁長時間測試後重組變慢。
  - 節點 debug 訊息限制保留行數，避免節點廣播太多造成 UI 狀態過大。
  - Mesh chat 訊息限制保留數量，避免長時間使用後列表越來越重。
  - Repository 自動清理逾時 pending response 與 completed seq，降低長時間連線記憶體累積。
  - PostList 無限捲動判斷補上狀態 key，搜尋模式不再觸發分頁載入。
- 版本：
  - `b0602d`
