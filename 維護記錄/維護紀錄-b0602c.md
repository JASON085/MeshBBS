# MeshBBS b0602c 維護紀錄

- 問題：MeshBBS App 讀取看板、文章列表、文章內容時仍偏慢，文章量增加後會更明顯。
- 修正：
  - Android App 改用 `LIST2` / `POSTS2` / `READ2` Mesh 專用短格式。
  - Server 對新版命令回傳短欄位 JSON，再交給既有 `zlib` + `MBBS2` 二進位分段壓縮傳送。
  - Server 分段回覆改為單次 private-DM，不再每段額外送 private broadcast 備援，減少約一半封包量。
  - Server 分段間隔由 1 秒降為 0.35 秒，維持排隊穩定性並縮短等待時間。
  - 舊版 `LIST` / `POSTS` / `READ` 格式仍保留，避免破壞 PC client 或舊測試流程。
- 版本：
  - `b0602c`
