# MeshBBS b0602o 維護紀錄
- 建置流程改為預設產生 signed release APK，降低安裝時被 Play Protect 視為一般 debug APK 的機率。
- 新增專案本機簽章：`android/MeshtasticBBS/signing/meshbbs-release.jks`。
- 新增 release 簽章設定：`android/MeshtasticBBS/signing/signing.properties`。
- `build_apk.ps1` 預設執行 release build，仍可用 `-BuildType debug` 建 debug APK。
- release build 關閉 minify，避免本機測試版被混淆/壓縮造成額外風險。
- 修正 release AAPT2 編譯 launcher PNG 失敗問題，將舊密度 PNG 重新產生為有效 PNG；主要圖示仍使用 adaptive icon。
- 本次版本號升為 b0602o，避免與既有 APK 重複。
- 已產生並驗證簽章 Client APK：`android/MeshtasticBBS/apk_archive/MeshBBS-b0602o.apk`。
- 已產生並驗證簽章 Server APK：`android/MeshtasticBBS/apk_archive/MeshBBS-server-b0602o.apk`。
- 注意：從舊 debug 簽章改裝 release 簽章時，Android 可能要求先解除安裝舊版；之後同一簽章版本可直接覆蓋更新。
