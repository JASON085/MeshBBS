# MeshBBS 維護紀錄

## APK 版本規則

- 每次製作 APK 前，先檢查 `android/MeshtasticBBS/apk_archive/` 與 `android/MeshtasticBBS/app/build/outputs/apk/debug/` 內既有版本。
- 新 APK 版本號不可與既有 APK 重複。
- 不要刪除舊版 APK，需保留在 `apk_archive/`。
- 目前已見版本：

## 目前修正發現的問題

- Android 登入按鈕曾被固定為 disabled，已改為帳號密碼都有輸入時可按。
- Android 看板解析原本只支援物件格式，Server Mesh LIST 回傳精簡陣列格式會造成解析錯誤，已補支援。
- 登入頁診斷曾直接顯示原始 payload，可能出現亂碼，已改為顯示摘要。
- 空看板頁面的「透過 Meshtastic 載入看板清單」屬舊版手動貼上流程，已移除。
- 登入頁的「透過 Meshtastic APP 傳送登入指令」屬舊版手動貼上流程，已移除。
- Android 管理功能原本只允許作者編輯/刪除，已補作者、版主、管理員判斷。
- Android 原本無法修改/刪除自己的回覆，已補每則本人回覆的修改與刪除操作。
- Server Mesh `EDIT` / `DEL` 原本只允許作者，已補版主與管理員權限。
- Server Mesh 原本沒有回覆編輯指令，已新增 `EDITREP`。
- Server 精簡 READ 回傳原本沒有 `board`，Android 無法判斷版主權限，已補回。
- 本機 Python 3.13 Tcl/Tk 在原安裝路徑下無法初始化，已建立 `C:\Users\JASON\.tcltk-python313` 並設定 `TCL_LIBRARY` / `TK_LIBRARY`。
- 專案 `.venv` 目前指向不存在的 Python 3.14，啟動工具已避開此壞環境。
- `啟動打包工具.bat` 已改為可停留顯示錯誤，且內建 Tcl/Tk 修復路徑。
- Android 登入頁的註冊 tab 曾缺少送出按鈕，已補上「註冊並登入」。
  - `b0602g`
- Android 登入頁已改為單一登入流程；註冊分頁移除，改在登入表單提示新用戶首次登入自動建立帳號。
