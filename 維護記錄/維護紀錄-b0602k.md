# MeshBBS b0602k 維護紀錄
- Android App 改為 client / server flavor 分流，避免把 Client 與 Server 功能混在同一個 APK。
- Client APK 恢復為原本單一功能入口，預設直接進入既有 MeshBBS Client 流程。
- Server APK 走獨立 flavor，仍維持 MeshBBS-server-版本號.apk 命名。
- build_apk.ps1 預設建置 Client；新增 build_server_apk.ps1 專門建置 Server。
- 本次版本號升為 b0602k，避免與現有 APK 重複。
- 修正 Android 專案內先前已損壞的 `BbsViewModel.kt`、`MeshtasticRepository.kt`、`ServerHostScreen.kt` 字串與語法錯亂，避免 client/server 分流後無法編譯。
